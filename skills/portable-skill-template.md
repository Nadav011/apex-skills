# Portable Skill Template

Use this file as the starting point when copying a skill into the public pack.

```md
---
name: skill-name
description: Short, public-facing description of the skill.
---

# Skill Name

## Purpose

Explain what the skill does in portable terms.

## Inputs

- What the skill expects from the user or workspace

## Outputs

- What the skill produces or verifies

## Notes

- Keep claims limited to what has been verified locally.
- Remove any repo-specific details before publishing.
```
